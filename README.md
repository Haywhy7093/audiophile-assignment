# Audiophile - Vite + React + Tailwind (scaffold)

This is a scaffolded frontend project that reproduces the Audiophile landing page design.

## Quick start

1. Install dependencies:
   ```
   npm install
   ```

2. Run development server:
   ```
   npm run dev
   ```

3. Build for production:
   ```
   npm run build
   ```
